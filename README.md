# Radiomics survival prediction

Source code for:

**A comparative study of survival prediction and feature selection/extraction methods for quantitative radiomics risk modelling**



## Overview

This repository provides R source code for evaluating survival prediction and feature selection/extraction methods in quantitative radiomics risk modelling.

The code implements multiple survival modelling approaches and feature selection strategies for comparative analysis of radiomics-based prognostic models.


## Contents

- `Radiomics_GitHub.R`: Main R script containing model training, feature selection, parameter tuning, and performance evaluation procedures.


## Requirements

R packages required for running the script include:

- survival
- glmnet
- randomForestSRC
- survivalsvm
- BhGLM
- keras
- tensorflow
- ggplot2
- survminer


## Usage

Modify the data paths and parameter settings according to your dataset before execution.
